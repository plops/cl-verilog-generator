- verilog operators
|                |                        |   |
| +,-            | addition, subtraction  |   |
| <<, >>         | shift                  |   |
| -              | negation               |   |
| ?:             | tertiary               |   |
| ~              | bit-wise negation      |   |
| ^              | bit-wise xor           |   |
| \vert, &       | bit-wise or, and       |   |
| \vert\vert, && | logical or, and        |   |
| !              | logical negation       |   |
| >>>            | arithmetic right shift |   |
| ==, !=         | equality, inequality   |   |
| <, <=          | less than (equal)      |   |
| >, >=          | greater than (equal)   |   |
|----------------+------------------------+---|
| *              | multiplication         |   |
| /, %           | division, remainder    |   |
